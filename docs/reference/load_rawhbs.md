# load_rawhbs

Function to load the Spanish Household Budget Survey (HBS).

## Usage

``` r
load_rawhbs(year, path, path_outputs)
```

## Arguments

- year:

  year of the HBS you want to load.

- path:

  Local path to the folder where the HBS's are stored. Not included in
  the package.

- path_outputs:

  path to save the results (RData)

## Value

a list with the 3 files of the HBS.
